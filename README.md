# Cat Pinterest

Небольшое SPA на React + TypeScript для просмотра изображений котиков из [TheCatAPI](https://thecatapi.com).
В приложении две страницы: все котики и избранные. Избранные сохраняются между сессиями.

## Что реализовано

- Лента всех котиков с бесконечной подгрузкой;
- Добавление/удаление котика в избранное;
- Отдельная страница с понравившимися котиками;
- Веселые гифки для состояния загрузки и ошибок.

## Ключевые решения

- **Архитектура через Context + hooks**: данные вынесены в отдельные провайдеры (`CatsProvider`, `FavoritesProvider`), UI-компоненты остаются простыми.
- **Бесконечный скролл**: реализован через `IntersectionObserver` (sentinel-элемент внизу списка) и защиту от параллельных запросов.
- **Устойчивость сетевых запросов**: используется `AbortController`, чтобы корректно отменять неактуальные запросы.
- **Хранение избранного**: лайкнутые котики хранятся в `localStorage`.
- **Навигация**: `HashRouter` для стабильной работы на GitHub Pages.
